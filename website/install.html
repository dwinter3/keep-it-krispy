Step 2: Install MCP Server

One-Line Install

curl -fsSL https://krispy.alpha-pm.dev/install.sh | bash

The installer will:

  1. Check prerequisites (Git, Node.js)
  2. Clone the repository to ~/keep-it-krispy
  3. Build the MCP server locally
  4. Ask for your User ID (from the dashboard)
  5. Configure Claude Desktop and/or Claude Code
Manual setup? See the MCP Setup Guide for step-by-step instructions.

Step 3: Configure Krisp Webhook

Connect your Krisp app to Keep It Krispy so transcripts sync automatically.

  1. Open Krisp desktop app
  2. Go to Settings â†’ Integrations â†’ Webhooks
  3. Webhook URL: Copy from your dashboard settings
  4. Request Headers: Click the + button to add a header:
    • Header Name: X-API-Key
    • Header Value: Your API key from the dashboard
  5. Select "Transcript created" as the trigger event
  6. Click "Update" to save
Why the API key header? This authenticates your webhooks so only your Krisp app can send transcripts to your account.

Troubleshooting

MCP Server Won't Connect

Check that Node.js 18+ is installed and the MCP server built successfully. Try running the installer again.

No Transcripts Showing

Verify your Krisp webhook is configured with the correct URL and API key. Check the dashboard for recent uploads.

Claude Can't Find Meetings

Make sure your User ID is set correctly in the MCP config. Restart Claude Desktop after changes.
