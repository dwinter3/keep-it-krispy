What It Does

Keep It Krispy captures your Krisp meeting transcripts and transforms them into a searchable knowledge base accessible directly from Claude.

Ask questions like:

"What did Caroline say about the legal timeline?"

Pull exact context from any meeting with any participant

"Summarize my calls from last week"

Get instant rollups of recent conversations

"Find meetings where we discussed Q1 roadmap"

Semantic search finds relevant context even without exact keywords

The Magic: Semantic Search

Search for "budget concerns" and find meetings where you talked about "cost overruns" or "we're spending too much" — no keyword matching, actual meaning.

Speaker names are indexed into the embeddings, so searching "meetings with Brian" works even if his name isn't in the transcript text.

The Stack

  • Krisp webhook → Lambda → S3 (you handle the hard part)
  • S3 event trigger → Processing Lambda that chunks transcripts
  • Bedrock Titan for embeddings (1024 dimensions)
  • S3 Vectors for similarity search (serverless vector DB)
  • DynamoDB for fast metadata queries
  • MCP server for Claude Desktop and Claude Code
  • Next.js dashboard on Amplify for web UI
