What It Does

Turn your Krisp.ai meeting transcripts into searchable, AI-queryable knowledge. Ask Claude about any past meeting.

Auto-Index Meetings

Every Krisp call is automatically captured, summarized, and indexed in the cloud.

Semantic Search

Find meetings by meaning, not just keywords. Search "budget concerns" and find "cost overruns".

Ask Claude Anything

"What did we decide about the timeline?" "Summarize my calls with Sarah this week."

Speaker Profiles

AI-enriched speaker bios from web search. Verify matches, add hints, track who you've met.

Topic Analysis

Auto-extract topics from conversations. Browse by topic to find related discussions.

Relationship Intelligence

Track companies mentioned in calls. See meeting history by company and speaker.

Web Dashboard

A full-featured web interface to browse, search, and manage your meeting memory.

Transcript Browser

View transcripts in iMessage-style chat bubbles or traditional format. See speaker talk time breakdown.

Speakers Directory

Browse everyone you've met. View profiles with AI-generated bios, meeting history, and topics discussed.

Advanced Search

Filter by date range, speaker, duration. Full-text search across all transcripts.

Bio Enrichment

One-click web search finds LinkedIn profiles. AI validates matches against your meeting context.

Privacy Controls

Mark transcripts as private, delete meetings, manage your data. Full control over your meeting history.

Company Tracking

See all meetings with a company. Track relationships across your network.

Requires Krisp.ai

Krisp.ai is a desktop app that provides real-time transcription for any meeting software (Zoom, Teams, Meet, etc.).

Why Krisp? Krisp works with any meeting platform, provides high-quality transcriptions, and has a webhook feature that sends transcripts to your infrastructure. You need Krisp installed to generate transcripts.

Uses Model Context Protocol (MCP)

MCP is Anthropic's standard for connecting AI assistants to external tools and data sources.

Claude Desktop

Add as an MCP server in Claude Desktop for native integration.

Claude Code

Use claude mcp add to add the server to your CLI.
